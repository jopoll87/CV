Bio

In 2012 I earned my Bachelors Degree in History and Spanish Teaching. I started working at Sunset Jr. High that fall teaching History part time. In the summer of 2013 I went back to school to earn my Masters Degree. I studied at Concordia University, Portland and earned my Masters Degree in Curriculum and Instruction with and emphasis in History Teaching. In 2017 the Administration at Sunset approached me and asked if I would be interested in teaching a Coding class. I had started to learn some programming on the side but not enough to teach, but I agreed. I spent the summer of 2017 taking classed and researching coding curriculum. I settled on Apple’s Everyone Can Code Curriculum. I loved this plan as students would be writing actual code and not doing “Block Coding.” There was also a good progression for a 3 year coding pathway. Since 2017 I have started teaching Swift, Python, and HTML/CSS.

In 2019 I started working at Apple Station Park as a Product Zone Specialist. I have the amazing opportunity to help people find the Apple device that will allow them to create amazing things for their families or their employers. I love being able to geek out with the customers over Apple’s amazing devices.

Teaching

  • Intro to Python: Grades: 7. An introductory programming course. Students will learn the basic of coding using the programming language, Python. Python is one of the fastest growing language in popularity and useability.
  • Swift-Learn to Code 1 & 2: Grades 8 & 9. Students will learn the basics of coding using the programming Language Swift. This course will help studens prepare to take Develop in Swift-Explorations. Swift is Apple's programming language. iPhone, iPad, Mac, AppleTV, and AppleWatch apps are programmed uisng Swift.
  • Develop in Swift-Explorations: (Pre-requisite: Swift-Learn to Code 1 & 2) In this course students will continue to develop their skills in programming in the Swift language. Students use the coding IDE Xcode. This is the same program that professional programmers use to create Apps.
  • U.S. History: Students will learn the history of their country. Starting with the Native Americans up through the year 1877, or the end of the time period called Reconstruction.
